🇹🇷 Türkçe

🎓 Teacher — Lesson prep + student tracking

Goal: Plan the weekly math lesson, keep attendance, and keep student files up to date.

BAIOS setup:

  • Project: matematik-8-a
  • Connected ARMs: file, dosya_oku, calendar, gmail_send
  • Connected skills: konu_aciklama, flashcard, gunluk_ozet
A typical day: 
baios> /proje sec matematik-8-a
baios> gelecek hafta trigonometri ders planı hazırla
  → konu_aciklama skill → docs/ders_plan_hafta14.md
baios> bu konudan 20 flashcard çıkar
  → flashcard skill → outputs/flashcards_trigonometri.csv
baios> ayşe'nin son sınav notunu dosyaya ekle 75
  → file arm → docs/ogrenciler/ayse.md güncel
baios> /otomasyon kur: her pazar 20:00 — hafta özetini bana e-posta at

📊 Accountant — Invoice analysis + VAT check

Goal: Check end-of-month invoices, verify VAT amounts, produce a report.

BAIOS setup:

  • Project: muhasebe-2026
  • ARMs: file, dosya_oku (PDF), sql (record db)
  • Flow: muhasebe_kontrol
A typical day: 
baios> /proje sec muhasebe-2026
baios> bu ay gelen faturaları docs/faturalar-nisan/ içine koydum, kontrol et
  → dosya_oku (PDF) → llm analiz → tutarsızlık uyarısı
  → sql arm → ay toplamı hesapla → rapor.md
baios> şirket X için nisan özeti hazırla
  → veri_analiz skill → outputs/sirket_X_nisan.md

Audit: Every action is recorded in ~/.baios/audit/ — an answer to the later question of who did what.

🧑‍💻 Developer — baios.dev dogfood

Goal: Manage the baios.dev website through BAIOS itself (the real story of this site).

baios> /proje sec baios-dev

baios> /skill calistir site_qa url=https://baios.dev

🎓 Researcher — Source scanning + wiki accumulation

Goal: Literature on "artificial intelligence + law" for a doctoral thesis, a continuously growing wiki.

BAIOS setup:

  • Project: doktora-tez-ai-hukuk
  • ARMs: brave_search, web_reader, file, notion
  • Skills: arastirma, url_ozet
  • Automatic wiki generation
A typical day: 
baios> bu akademik makale URL'sini oku ve özetle: https://arxiv.org/abs/...
  → web_reader + llm_ozet → outputs/kaynak_N.md
baios> önemli bulgulardan biri: "AI karar verme şeffaflık sorunu"
  → /wiki olustur ai-karar-seffafligi
baios> geçen ay özetlediğim tüm kaynakları göster
  → file arm → docs/kaynaklar/ listele
baios> ilgili wiki sayfalarını LLM context'e ekle ve tez 3. bölümü için taslak hazırla

The benefit: Every source is permanently added to your knowledge. A month later, even if the LLM changes, the accumulation is preserved.

The Common Thread: Discipline

In all six scenarios, a different profession, a different job. What's common: 1. Project — BAIOS keeps every job alive inside a project. 2. Discipline — The working method lives in a file, repeatable. 3. Approval — Risky operations ask for approval. 4. Context — The LLM always looks at the current state. 5. Accumulation — Each project accumulates its own knowledge, the wiki grows.

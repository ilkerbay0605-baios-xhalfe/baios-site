TL;DR: Step by step from installation to running your first skill, for individual users.

Quick Start

🇹🇷 Türkçe

In ten minutes you'll install BAIOS, open your first project, and run your first skill.

1. Installation (1 min)

# Early version — install from GitHub (PyPI release coming soon)
git clone https://github.com/ilkerbay0605-baios-xhalfe/baios
cd baios && ./run.sh start

On the first run it asks a few short personalization questions (name, language). Once you confirm, the main menu opens.

Web interface: 

baios serve
# → http://localhost:8000

2. Choose an LLM Provider (1 min)

At least one LLM provider is required. Three options:

OpenAI (if you have an API key): 

baios config set llm.provider openai
baios config set openai.api_key sk-...

Anthropic: 

baios config set llm.provider anthropic
baios config set anthropic.api_key sk-ant-...

Local Ollama (no internet, free): 

# In a separate window:
ollama serve
ollama pull llama3.2

baios config set llm.provider local

If you have sensitive data, prefer **local**. If you want speed and accuracy, use OpenAI/Anthropic.

3. Your First Project (2 min)

Web UI: 1. Left menu → Projeler → Yeni 2. Name: denemem 3. Description: BAIOS'u denemek için 4. Oluştur

CLI: 

baios> /proje yeni denemem "BAIOS'u denemek için"
baios> /proje sec denemem

The project directory is created: ~/.baios/projects/denemem/.

4. Generate a Working Discipline (2 min)

What is the project for? Tell the Discipline Engine and let the LLM prepare a workflow tailored to you.

Web UI: 1. On the project page, go to the Çalışma Disiplini section 2. Goal: "Keep a weekly budget table and produce an end-of-month summary" 3. Üret

The result is discipline_v1.md — a step-by-step method in 3-5 phases, with recommended ARMs and skills.

5. Your First Skill (2 min)

Let's try a simple skill — arastirma:

Web UI: 1. Left menu → Kütüphane → Skill 2. Find arastirma → Çalıştır 3. konu: en iyi sabah rutinleri 4. Çalıştır

Three steps run: 1. Web search 2. LLM summary 3. Save to file → ~/.baios/projects/denemem/outputs/arastirma_*.md

6. Install an ARM (optional, 1 min)

If you want to read Gmail, install the gmail ARM:

Web UI: 1. Left menu → Kütüphane → ARM 2. gmail → Kur 3. Google account OAuth flow 4. Done

Now gmail is available in skills.

7. Talk from Chat (1 min)

Now you can speak in natural language:

baios> bugünkü e-postalarımı özetle
  → gmail arm + llm_ozet skill
baios> istanbul hava durumu
  → weather arm
baios> bu araştırma dosyasından 10 flashcard çıkar
  → flashcard skill

BAIOS reads your intent and runs the appropriate ARM/Skill. For risky operations it asks for approval.

Troubleshooting

Installation fails: 

# Requires Python 3.12+ and git
git clone https://github.com/ilkerbay0605-baios-xhalfe/baios
cd baios && ./run.sh start

Web UI won't open: Port 8000 may be used by another application. baios serve --port 8080

LLM error: 

baios config show       # see the current provider
baios config test llm   # connection test

Log: 

tail -f ~/.baios/logs/baios.log

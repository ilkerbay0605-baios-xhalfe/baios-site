Skill — Repeatable Workflows

Content Creation

  • arastirma — search a topic on the web → summarize with the model → save
  • blog_taslak — topic + audience → 800–1200 word draft
  • blog_post_taslak — (SM5) a more refined blog draft
  • icerik_uretim — multi-step content creation
  • landing_copy_revize — revise landing copy for the audience
  • url_ozet — URL → markdown → summary
  • gorsel_prompt — generate an image prompt for DALL-E

Professional Work

  • muhasebe — invoice/CSV → analysis → report
  • fatura_olustur — line items → invoice PDF
  • harcama_takip — add an expense item + category
  • veri_analiz — read CSV → basic statistics → chart description
  • rakip_izleme — competitor site → change detection
  • fiyat_takip — URL list → track prices → alert

Daily

  • haber_ozet — RSS/feed → model summary → email
  • gunluk_ozet — summarize the day's emails/messages
  • toplanti_ozet — summarize a transcript
  • takip_mail — customer → follow-up email draft
  • ilac_hatirlatma — timed reminder
  • dosya_organize — organize a folder by type

Developer / Education

  • flashcard — text → Anki card set
  • konu_aciklama — topic → graded explanation
  • git_changelog — git log → user-friendly change list

baios.dev (SM5)

BAIOS

A controlled working discipline that runs in human language, model-independent.

🇹🇷 Türkçe

We used to have notebooks, pens, and tools on our desk. Then came folders and windows. Now, with AI, a new working desk is taking shape — and at this desk we finally speak the same language. BAIOS is not a doer that runs off on its own; it is a layer that works between the model and the human, carrying out controlled, recorded, and reversible actions.

An assistant built for your profession — KVKK-compliant, your data stays with you. From lawyers to accountants, SMB owners to researchers — everyone's work is different; BAIOS works with each in their own language.

Not a rival to Anthropic Claude, a partner-compatible complement. Author once, run forever — own it, don't rent it.

Core Principles

  • **Language first** — No commands to memorize. Write or say what you want; BAIOS understands.
  • **No action without approval** — Deleting files, deploy, API calls… all ask for your approval.
  • **Simplicity** — A 5-line answer, not 50.
  • **Context is everything** — User, project, files, wiki knowledge — all sent to the model automatically.
  • **Model independence** — OpenAI, Anthropic, local Ollama. BAIOS is independent of the model.

How Does BAIOS Work?

Project discipline: Each project lives with its own context, materials, and connected ARMs/skills. BAIOS knows the project's structure and suggests smart actions.

Components:

  • ARM — Adapters that connect to the outside world (Gmail, GitHub, files, shell…)
  • Skill — Repeatable workflow templates (research → summarize → save)
  • Flow — Large flows with conditions, loops, multiple skills
  • Automation — Scheduled or triggered tasks
  • MCP — BAIOS is both an MCP server and client (connects to any MCP server)

What Is It?

omgunbay — the official BAIOS chatbot. It works with RAG (retrieval-augmented generation) over all BAIOS documentation (BKS, PROJE_PLANI, BAIOS_PRINCIPLES, tutorial, resources).

In plain words: ask anything about BAIOS, and it looks at the written documents and answers. It does not make things up; when it does not know, it says "I don't know."

Where Can I Find It?

1. The chat window at the bottom right of baios.dev

On every page of this site (very soon) — open the chat button at the bottom right and start. No sign-up required.

2. Telegram

@omgunbay_bot — start the conversation on Telegram with /start.

3. Inside BAIOS

In your own BAIOS installation: 

baios> /chat omgunbay "yeni skill nasıl yazılır?"

How Does It Work?

Knowledge base:

  • BAIOS_PRINCIPLES.md — philosophy, principles
  • PROJE_PLANI.md — roadmap, phase plans
  • agents/BKS.md — version, status
  • site/docs/* — this site itself
  • baios_bireysel_cli_tutorial.md — CLI tutorial
  • Source code comments (partial)
Process: 1. Converts your question into an embedding 2. Finds the most relevant passages in the knowledge base 3. Tells the LLM "answer based on these passages" 4. States which source it drew on in the answer

Assurance:

  • Says "I don't know" for topics outside the knowledge base
  • Cites sources
  • The answer is verifiable

What It Does Not Do

  • **It does not analyze project data** — it only talks about BAIOS.
  • **It does not give live weather / news** — the knowledge base is static.
  • **It does not store conversation history** (optional — can be stored in enterprise mode).
  • **It does not take action** — it only gives a written answer. You cannot say "open a PR"; you need to go to the BAIOS CLI/web for that.

Feedback

If omgunbay gives a wrong answer:

  • The 👎 button inside the chat
  • GitHub issue: [omgunbay] label
  • What the correct answer should have been — write it, so the knowledge base gets updated

Developer Note

The omgunbay source code is in baios/omgunbay.py. It is a simple RAG implementation — it is re-indexed when the knowledge base is refreshed.

Extending the knowledge base: When you add a new tutorial or a new document, the index is updated with baios omgunbay rebuild.

Basics

What is BAIOS, in one sentence? A controlled work discipline that runs in human language and is LLM-independent.

Is it an assistant that acts on its own? No. BAIOS is a layer — you do the thinking, BAIOS applies it with discipline. Details

Is it paid? MIT license, free and open source. LLM calls may be paid depending on the provider (OpenAI/Anthropic). If you use local Ollama, there is no charge at all.

Turkish or English? Both. Documentation is Turkish-first; in-code messages are mostly Turkish. The i18n infrastructure is ready.

Installation

How do I install it? 

# early version — from GitHub (PyPI release coming soon)
git clone https://github.com/ilkerbay0605-baios-xhalfe/baios
cd baios && ./run.sh start
Tutorial

Does it work on Windows? We recommend WSL2. Pure Windows paths are not tested; please report issues.

Python version? 3.12+.

Is Ollama required? No. If you have an OpenAI/Anthropic API key, that is enough. Ollama is only for a local LLM.

Project & Discipline

What is a project? A purpose-oriented set of files + context + a connected collection of ARMs/Skills. It lives in the ~/.baios/projects// directory. Architecture

How many projects can I have? No limit. You can open hundreds of projects.

Can I move a project? Yes — baios proje export produces a ZIP, which can be loaded on another machine with baios proje import.

Is project discipline mandatory? No, it is optional. But very useful — the LLM's suggestions to you become project-specific.

ARM & Skill

What is an ARM? An adapter — the Python module that connects BAIOS to external services (Gmail, GitHub, file, shell). Catalog

How many ARMs are there? 28 core + 8 optional in the community store.

Difference between a skill and an ARM?

  • ARM: does a single job (one API call, one file operation)
  • Skill: a template that combines multiple steps
How do I write my own skill? With a JSON file. Developer guide

Can I give my skill JSON to a friend? Yes — a single file, copy-paste and it works. You can also upload it to the community store.

LLM

Which LLMs are supported? OpenAI (GPT-4/4o/5), Anthropic (Claude Opus/Sonnet), Ollama (Llama 3, Mistral, Qwen), Local_4K (small models).

Switching providers? 

baios config set llm.provider anthropic
The conversation continues from where it left off — the context is preserved.

Does the LLM have a hallucination problem? BAIOS's current capabilities are injected into the context (SM-FIX1). The LLM won't say "this skill doesn't exist" — it knows whether it is in the list or not. Even so, human approval is taken on every important output.

What is fallback? If the priority provider goes down, the next one is tried. Uninterrupted service.

Data & Privacy

Where is my data? Local: ~/.baios/. It goes to an external service only at the moment of an LLM call (to OpenAI if you chose OpenAI).

Is there a BAIOS server, does it take my data? No. BAIOS is entirely local. Telemetry is optional and off by default.

Are my API keys safe? Encrypted in the vault with Fernet + PBKDF2. The master password is yours.

Would it accidentally run rm -rf? No. The shell ARM uses an allowlist; rm is on the blocklist. In general, the rule is "no risky action without approval."

Automation

Is it like cron? Yes + more. Time, webhook, file change, RSS, MQTT — all can be triggers.

What if there is an error? Every run is recorded in the audit. If there is an error, an email/Telegram notification (configurable).

Stop an automation? 

baios> /otomasyon durdur

Integration

Are Slack/Discord/Telegram supported? Yes, all 17 channels. Bot setup in order at catalog → channels.

Can it connect to Notion? Yes, with the notion ARM.

Can an MCP server be added? Yes, Many MCP tools can be connected to BAIOS as ARMs. MCP guide

Triggering GitHub Actions? workflow_dispatch with the deploy ARM.

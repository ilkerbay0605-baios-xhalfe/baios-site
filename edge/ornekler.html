🏠 Akıllı Ev: "Sabah Rutini"

Donanım: Raspberry Pi 4, Home Assistant, Zigbee köprüsü

Kurgu:

  • Proje: ev
  • ARM: homeassistant, calendar, mqtt
  • Otomasyon: "Sabah alarmı" + "akşam kilitleme"
Kural (doğal dil): 
her hafta içi sabah 7:00:
  1. yatak odası perdesini aç
  2. kahve makinesini çalıştır (su seviyesi %30'un üstündeyse)
  3. bana bugünün hava durumunu ve takvim özetini söyle

BAIOS nasıl yapar:

  • Cron tetikler → homeassistant (perde) → homeassistant (kahve + sensör) → weather + calendar → TTS
Neden BAIOS: Hepsi tek yerden, aynı dilde. Bozulursa komutu değiştir, kod yazma.

📊 Sensör Veri: "Ev Sağlık Özeti"

Donanım: 5× ESP32 sensör (sıcaklık, nem, ses, CO2, ışık) + Raspberry Pi 4 brain

Kurgu:

  • ESP32'ler MQTT'ye yayınlıyor (saniyede değer)
  • Pi üstünde Mini-BAIOS MQTT dinliyor
  • SQLite'a kaydediyor
  • Her sabah 8:00 günlük özet LLM'e gidiyor
Rapor örneği:
"Dün gece salonda CO2 seviyesi 1400 ppm'e çıktı (03:00-05:00). Yatak odasında ses 2 saat boyunca 45dB üstündeydi. Önerim: havalandırmayı akşam 22:00'de 15 dk aç."

BAIOS skill: sensor_gun_ozet (edge için özel)

🤖 Robot: "Oda Temizleme Komutları"

Donanım: Jetson Orin Nano + LIDAR + ROS 2

Kurgu:

  • Operatör sesli komut veriyor
  • BAIOS STT → niyet analizi → ROS 2 goal mesajı
  • Risk kontrolü (kırılacak eşya var mı, şarj yeterli mi)
  • Aksiyon → audit'e kayıt
Komut örneği: 
operatör: "mutfağın sağ tarafından başla, 20 dakika içinde salona geç"
BAIOS: 
  - rota_planla(start="mutfak_sag", end="salon", sure=20dk)
  - şarj=%85 → OK
  - kırılacak_eşya tespit → "dikkat: vazo x=3.2 y=1.8"
  - onay? → operator OK
  - ROS goal publish

🔧 Maker: "Konuşan Akvaryum"

Donanım: ESP32 + OLED + sıcaklık/pH sensörü + Mini-BAIOS (Pi Zero üstünde)

Kurgu:

  • Akvaryum sensörlerini okur
  • Pi Zero'daki Mini-BAIOS kullanıcının balıklarının türünü/bakım kurallarını bilir
  • Anomali durumunda e-posta/Telegram
Kullanıcı sohbeti (telefondan): 
user: "bugün su nasıl?"
BAIOS: "Sıcaklık 24.8°C (ideal 24-26), pH 7.2 (ideal 7.0-7.5). 
       Nitrit seviyesi dün 0.1'e çıktı, bugün 0 — güzel. 
       3 gündür yem değiştirmedin, hatırlatayım mı?"

🚗 Sürücü Asistanı (araç içi)

Donanım: Jetson Nano + CAN-bus adaptör + mikrofon + hoparlör

Kurgu:

  • Araç sensörlerine (yakıt, motor sıcaklık, hız) erişim
  • BAIOS sürücü ile doğal dil konuşur
  • Trafik verisi (offline harita veri) + hava durumu
Senaryo: 
user: "eve gidiş için yakıt yeter mi?"
BAIOS: "Evet, kalan yakıtla ~280 km. Eve 95 km. 
       Yolda 2 benzin istasyonu var, ihtiyaç yok. 
       Hava bozuyor — yağmur 35 dk sonra başlar, dikkat."

⚙️ Endüstriyel: Makine Bakım Tahmini

Donanım: Fabrika makine PLC + Modbus gateway + Raspberry Pi cluster + Mini-BAIOS

Kurgu:

  • PLC sensör verisi → MQTT
  • BAIOS saatlik anomali tespiti (basit ML)
  • Bakım mühendisine Telegram bildirim + dosya log
Kullanıcı: 
mühendis: "makine 3 son hafta nasıl?"
BAIOS: "Titreşim 3. hafta %18 arttı. Son 24 saatte lokal pik 
       %32. Bu trend devam ederse 9-12 gün içinde rulman 
       değişimi gerekecek. Stokta yedek var mı kontrol edeyim mi?"

Ortak Özellik: Hepsi Aynı BAIOS

Altı senaryoda da aynı core çalışıyor:

  • Proje disiplini
  • Context katmanları
  • Onay zinciri
  • Audit log
Fark sadece hangi ARM'lar aktif ve LLM nerede (yerel vs bulut).

**Durum:** Yakında. Aktif araştırma / erken prototipleme. Beta duyurusuna [buradan](beta-kayit.html) kayıt ol.

Vizyon

BAIOS bir PC katmanı değil. ESP32'ye sıkışmış bir sensör-cihaz, Raspberry Pi üzerinde bir robot, akıllı ev kontrolcüsü — hepsi aynı çalışma disiplini ile konuşsun.

  • **Doğal dil:** "Salondaki sıcaklık 25 dereceyi geçerse pencereyi aç."
  • **Context:** Cihaz kendi sensörlerini, geçmiş değerlerini, kural setini bilir.
  • **Bağımsız:** Tüm LLM çağrıları yerel (küçük model) veya koşullu bulut.
  • **Kontrollü:** Riskli aksiyonlar (motor çalıştır, kapı aç) onay zinciri ile.

Mini-BAIOS Profili

Mimari: Tam BAIOS'un alt kümesi — core + sınırlı ARM seti.

Hedeflenen kaynaklar:

  • ESP32 sınıfı: 8MB flash, 512KB RAM — sadece kural motoru + MQTT köprüsü, LLM dışarıda.
  • Raspberry Pi 4 sınıfı: 1GB RAM — core + yerel 4K LLM + 6-8 ARM.
  • Jetson Nano / Orin sınıfı: Tam BAIOS + yerel büyük LLM.
Seçili ARM'lar (planlı):
  • mqtt — sensör okuma/yazma
  • homeassistant — akıllı ev REST
  • http — genel
  • shell (kısıtlı whitelist)
  • file (sadece belirli dizinler)
  • sql (SQLite sensör kaydı)
  • rss (opsiyonel)

Kullanım Senaryoları

Akıllı Ev Kontrol

Raspberry Pi + Home Assistant köprüsü. "Sabah 7'de alarm duyulduğunda perdeleri aç, kahve makinesini çalıştır" gibi dil tabanlı kurallar.

Sensör Veri Birikimi

ESP32 cluster → MQTT → Mini-BAIOS → SQLite → günlük özet LLM. Her sabah "dün evin ortalama nem/sıcaklık/ses seviyesi" raporu.

Robot Komut Köprüsü

Robot işletim sistemi + BAIOS. Operatör "yerde sol alt köşeyi temizle" diye komut verir, BAIOS komutu ROS mesajlarına çevirir, risk değerlendirmesi yapar.

Maker & Hobi

ESP32 + OLED ekran + sensör + BAIOS Mini. Evde her cihazın "açıklaması" olur — cihaza sor, cihaz cevaplar.

Tutorial

🇬🇧 English

Baştan sona BAIOS rehberi. 45 dakikada tamamlayınca kendi iş akışını kurmuş olursun.

0. Başlamadan

İhtiyaç:

  • Python 3.12+
  • Bir LLM provider (API key VEYA yerel Ollama)
  • 15 dakika odak
Sonuç: Kendi projesi, bağlı ARM'ları, çalışan bir skill'i olan bir BAIOS kullanıcısı olmuş olacaksın.

Bölüm 1 — Kurulum (5 dk)

1.1 BAIOS'u yükle

# erken sürüm — GitHub'dan (PyPI yayını yakında)
git clone https://github.com/ilkerbay0605-baios-xhalfe/baios
cd baios && ./run.sh start

İlk açılışta kişiselleştirme sorar:

  • İsim
  • Dil (Türkçe önerilen)
  • Tercih ettiğin iletişim üslubu (dostça / resmi / teknik)

1.2 LLM provider seç

Yerel (ücretsiz, internet gerekmez): 

# Yeni terminal:
ollama serve
ollama pull llama3.2

# BAIOS terminal:
baios config set llm.provider local

OpenAI (hız + kalite): 

baios config set llm.provider openai
baios config set openai.api_key sk-...

Anthropic (uzun context): 

baios config set llm.provider anthropic
baios config set anthropic.api_key sk-ant-...

1.3 Test et

baios> merhaba

Konuşma başlamalı. Web UI için: 

baios serve
# → http://localhost:8000

Bölüm 2 — İlk Proje (5 dk)

2.1 Proje oluştur

Amacına göre bir proje düşün. Örneğin "haftalık bütçe takibi" veya "matematik dersleri" veya "blog yazı havuzu".

baios> /proje yeni butce "Aylık gelir-gider takibi"
baios> /proje sec butce

Veya web UI'den Projeler → Yeni.

2.2 Proje dosyaları

Proje dizini otomatik oluştu: 

~/.baios/projects/butce/
├── project_context.md       ← amaç, kurallar
├── project_tmp_context.md   ← geçici notlar
├── project_config.json      ← bağlı ARM/Skill
├── docs/                    ← materyallerini koy
└── outputs/                 ← skill çıktıları

2.3 Materyal ekle (opsiyonel)

Bütçe Excel'in varsa: 

cp ~/bütçe.csv ~/.baios/projects/butce/docs/

2.4 Disiplin üret

Web UI: Proje sayfası → Çalışma Disiplini → Hedef yaz → Üret

CLI: 

baios> /disiplin uret "Aylık bütçe tutmak ve ay sonu özet çıkarmak"

Sonuç discipline_v1.md — LLM sana özel iş akışı hazırladı.

Bölüm 3 — İlk ARM Kurulumu (5 dk)

3.1 Neye ihtiyacın var?

Düşün: projen hangi dış servise ihtiyaç duyar?

  • Gmail okumak mı? → gmail ARM
  • Web'den bilgi çekmek mi? → brave_search ARM
  • Hava durumu mu? → weather ARM
  • Sadece dosyalarla çalışmak mı? → ARM gerekmez (file ARM otomatik var)

3.2 ARM kur

Web UI: Kütüphane → ARM sekmesi → arayı bul → Kur

CLI: 

baios> /arm kur weather
# → API key sorar → OpenWeatherMap'ten ücretsiz al: https://openweathermap.org/api

3.3 Projeye bağla

baios> /proje bagla arm weather

Artık proje context'inde weather ARM mevcut — LLM bunu bilir.

Bölüm 4 — İlk Skill Çalıştırma (10 dk)

4.1 Skill seç

Başlangıç için arastirma basit ve güzel:

  • Bir konu verirsin
  • Web'de arar
  • LLM özetler
  • Dosyaya kaydeder

4.2 Çalıştır

Web UI: Kütüphane → Skill → arastirma → Çalıştır

  • konu: seni ilgilendiren bir şey, örn. "hangi bitkiler mutfakta yetişir"
CLI: 
baios> /skill calistir arastirma konu="hangi bitkiler mutfakta yetişir"

4.3 Sonucu incele

Çıktı proje dizininde: 

~/.baios/projects/butce/outputs/arastirma_hangi_bitkiler_mutfakta_yetisir.md

Web UI'de Projeler → butce → Dosyalar sekmesinden açabilirsin.

Bölüm 5 — Doğal Dil Konuşma (5 dk)

Artık komut ezberlemeyi bırak, doğal konuş:

baios> istanbul hava nasıl bugün?
baios> bütçe CSV'sinden toplam harcamayı hesapla
baios> "yeni teklif" ile başlayan mailleri listele (gmail ARM kuruluysa)
baios> bu araştırma dosyasından 5 madde özet çıkar

BAIOS niyeti okur, doğru ARM/Skill'i seçer, riskli işlemde onay ister.

Bölüm 6 — İlk Otomasyon (10 dk)

Tekrarlanan bir işi otomatikleştir.

6.1 Otomasyon tanımla

baios> /otomasyon kur
  Tetikleyici: her pazar 20:00
  Aksiyon: bütçe özeti çıkar + e-posta at (bana)

6.2 Çalışmasını gör

baios> /otomasyon listele

İlk pazar 20:00'de otomatik çalışır. Çıktıyı e-posta alırsın.

6.3 Değiştir / durdur

baios> /otomasyon durdur butce_ozet_haftalik
baios> /otomasyon sil butce_ozet_haftalik

Bölüm 7 — Wiki (5 dk)

Projede biriken bilgiyi kalıcı hale getir:

baios> /wiki olustur butce-kategoriler

BAIOS proje içeriklerini + konuşma geçmişini inceleyip bir wiki sayfası hazırlar. Her LLM çağrısında ilgili sayfalar otomatik context'e gider.

Birikim birikir — projen ne kadar eskidse o kadar akıllı.

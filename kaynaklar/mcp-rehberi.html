MCP — Model Context Protocol

🇬🇧 English

BAIOS hem MCP sunucusu hem MCP istemcisi. Bu ne demek, ne işe yarar?

MCP Nedir?

Model Context Protocol — Anthropic'in önerdiği, LLM araçlarının standart şekilde sunucu-istemci iletişimi yapmasını sağlayan protokol.

Basit benzetme:

  • Ethernet cihazları fiziksel olarak bağlar.
  • HTTP yazılımları web üzerinden konuşturur.
  • MCP, LLM uygulamaları ile araçları/verileri aynı protokolde konuşturur.
Sonuç: Bir MCP sunucusu yazan (Slack, Jira, Figma) onu bütün MCP-uyumlu uygulamaların (Claude Desktop, VS Code, BAIOS) kullanmasını sağlar. Çok sayıda MCP sunucusu mevcut, her gün artıyor.

BAIOS'un MCP Rolü

Sunucu olarak

BAIOS kendisini MCP sunucusu gibi sunar — dışarıdan başka araçlar BAIOS'un ARM/Skill/Flow'larına erişebilir.

Senaryo:

  • Claude Desktop'tan "İlker'in baios-dev projesinde PR aç" dersin
  • Claude Desktop BAIOS'un MCP sunucusuna bağlanır
  • BAIOS'un github ARM'ı çalışır → PR açılır

İstemci olarak

BAIOS dış MCP sunucularına bağlanır. Bir MCP sunucusu ekleyince o tool otomatik ARM olur.

Senaryo:

  • Figma MCP sunucusunu ekliyorsun (npx @figma/mcp-server)
  • BAIOS algılıyor → figma ARM kendiliğinden geliyor (api_figma olarak)
  • Skill'lerinde arm:api_figma kullanabilirsin

MCP Sunucu Ekleme

1. MCP sunucusunu kur

Her MCP sunucusunun kendi dokümantasyonu var. Örnek — GitHub MCP:

npm install -g @modelcontextprotocol/server-github

2. BAIOS'a tanıt

Web UI: Ayarlar → MCP Sunucuları → Ekle

CLI: 

baios> /mcp ekle github-resmi "npx -y @modelcontextprotocol/server-github" stdio
  Environment variable:
  GITHUB_TOKEN=ghp_...

3. ARM olarak görün

baios> /arm listele
# api_github_resmi — Auto ARM (MCP: github-resmi)

Artık skill'lerinde kullanabilirsin: 

{
  "action": "arm:api_github_resmi",
  "params": {"query": "list my repositories"}
}

Popüler MCP Sunucuları

Uyumluluk testi aktif. Şu an %100 test edilen sınırlı; aşağıdaki listede olanları **güvenle dene**:
  • **filesystem** — yerel dosya işlemleri (BAIOS'un kendi `file` ARM'ına alternatif)
  • **github** — GitHub API
  • **slack** — Slack workspace
  • **google-drive** — GDrive
  • **postgres** — PostgreSQL
  • **brave-search** — web arama
  • **puppeteer** — web scraping
Güncel liste: [modelcontextprotocol.io/servers](https://modelcontextprotocol.io)

BAIOS'u Dışarıdan Kullan

Claude Desktop'a BAIOS'u MCP sunucusu olarak ekle:

1. BAIOS MCP sunucu modunu başlat: 

baios mcp-server --port 7777

2. Claude Desktop config'te: 

{
  "mcpServers": {
    "baios": {
      "command": "baios",
      "args": ["mcp-server", "--stdio"]
    }
  }
}

3. Claude Desktop'ta: 

Bana baios-dev projesinde son 10 commit'i özetler misin?
→ Claude, BAIOS'un git_ops ARM'ını kullanır

Güvenlik

  • MCP sunucuları **yerel** çalışır (stdio) veya TLS ile.
  • BAIOS, MCP çağrılarını aynı onay zincirinden geçirir (riskli işlem → onay).
  • Audit log'a hem MCP'den gelen istek hem BAIOS'un cevabı kayıt.
  • Zararlı MCP sunucusu? BAIOS tarafında **yalnızca izinli eylemler** geçer.

Katkı

Kendi MCP sunucunu yazdıysan ve BAIOS uyumluluk testi istersen:

  • GitHub Issue: [mcp-compat] etiketiyle aç
  • Örnek yapılandırma + birkaç test komutu paylaş
  • Test sonucu repo'ya ekleriz

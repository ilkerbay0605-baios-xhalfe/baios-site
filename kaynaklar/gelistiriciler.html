Kod İndir

git clone https://github.com/ilkerbay0605-baios-xhalfe/baios
cd baios
pip install -e .

Python: 3.12+

Geliştirme ortamı: Linux / macOS (Windows: WSL önerilen).

Yeni Skill Yaz

1. JSON dosyası oluştur:

baios/skills/templates/ornek_skill.json

{
  "name": "ornek_skill",
  "label": "Örnek Skill",
  "description": "Konuyu araştırır + özetler + dosyaya kaydeder.",
  "category": "arastirma",
  "required_arms": ["brave_search"],
  "input_params": ["konu"],
  "steps": [
    {
      "action": "arm:brave_search",
      "params": {"query": "{konu}"},
      "description": "Web arama"
    },
    {
      "action": "llm:Sonuçları 5 maddede Türkçe özetle.",
      "use_previous": true,
      "description": "LLM özet"
    },
    {
      "action": "file:outputs/ornek_{konu}.md",
      "use_previous": true,
      "description": "Kaydet"
    }
  ]
}

2. Test et:

python -m pytest tests/test_site_qa_skill.py -v  # yapısal test örneği
curl -X POST http://localhost:8000/api/skills/ornek_skill/run \
     -H 'Content-Type: application/json' \
     -d '{"params":{"konu":"python test patterns"}, "project":"denemem"}'

3. Belge güncelle:

  • agents/EZME_KORUMA.md skill listesine ekle
  • PROJE_PLANI.md skill sayısını güncelle
Adım tipleri:
  • arm: — ARM çalıştır (config ARM'ın kendi kurulumundan)
  • llm: — LLM ile işle (use_previous: true ile önceki çıktıyı prompt'a ekle)
  • file: — kaydet (yol proje bağlamında ise proje/outputs'a gider)
  • skill: — başka skill çağır
Parametreler: {konu} gibi placeholder'lar. Dosya adlarında temizlenir (slugify).

Yeni ARM Yaz

Hızlı yol: Builder endpoint (SM1.5):

curl -X POST http://localhost:8000/api/arms/create \
     -H 'Content-Type: application/json' \
     -d '{
       "name": "my_arm",
       "label": "My ARM",
       "description": "My custom ARM",
       "type": "local",
       "install_level": 0
     }'

Bu, üç şeyi yapar:

  • baios/arm/my_arm_arm.py stub handler oluşturur
  • baios/arm/catalog.json'a kayıt ekler
  • baios/arm/handlers.py'ye register bloğu ekler
  • Canlı _HANDLERS dict'ine register eder (restart gerek yok)
Stub'u doldur:

# baios/arm/my_arm_arm.py

def handle_my_arm(action: str, config: dict, query: str):
    from baios.arm.handlers import HandlerResult
    
    # action: "search", "create", "list" vb. — skill şablonunda belirlenir
    # config: ~/.baios/arm_configs/my_arm.json içeriği
    # query: kullanıcı sorusu
    
    if action == "search":
        result = f"Aradığın: {query}"
        return HandlerResult(True, result)
    
    return HandlerResult(False, "", f"Bilinmeyen aksiyon: {action}")

Test yaz:

# tests/test_my_arm.py
def test_my_arm_search():
    from baios.arm.handlers import execute_arm
    r = execute_arm("my_arm", "search", {}, "test")
    assert r.success is True

Yeni Flow Yaz

Flow = birden fazla skill/ARM'ı koşullu çalıştırır.

baios/skills/flows/my_flow.json

{
  "name": "my_flow",
  "description": "Önce araştır, sonra blog yaz, sonra QA yap",
  "steps": [
    {
      "id": "s1",
      "type": "skill",
      "skill_name": "arastirma",
      "params": {"konu": "{konu}"},
      "save_as": "rapor"
    },
    {
      "id": "s2",
      "type": "skill",
      "skill_name": "blog_post_taslak",
      "params": {"konu": "{konu}", "okuyucu": "genel"},
      "save_as": "blog"
    }
  ]
}

Koşullu adım:

{
  "id": "kontrol",
  "type": "condition",
  "condition": "Blog taslak kalitesi yeterli mi? {blog}",
  "if_true": "yayinla",
  "if_false": "yeniden_yaz"
}

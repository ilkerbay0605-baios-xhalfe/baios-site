Temel

BAIOS nedir, tek cümleyle? İnsan dilinde çalışan, LLM bağımsız, kontrollü bir çalışma disiplini.

Ajan mı? Hayır. BAIOS bir katmandır — sen düşünürsün, BAIOS disiplinli şekilde uygular. Detay

Ücretli mi? MIT lisansı, ücretsiz ve açık kaynak. LLM çağrıları provider'a göre ücretli olabilir (OpenAI/Anthropic). Yerel Ollama kullanırsan hiç ücret yok.

Türkçe mi, İngilizce mi? Her ikisi. Dokümantasyon Türkçe öncelikli; kod içi mesajlar çoğunluğu Türkçe. i18n altyapısı hazır.

Kurulum

Nasıl kurulur? 

# erken sürüm — GitHub'dan (PyPI yayını yakında)
git clone https://github.com/ilkerbay0605-baios-xhalfe/baios
cd baios && ./run.sh start
Tutorial

Windows'ta çalışır mı? WSL2 öneriyoruz. Saf Windows yolları test edilmedi, sorun bildirin.

Python sürümü? 3.12+.

Ollama şart mı? Hayır. OpenAI/Anthropic API key'in varsa o yeter. Sadece yerel LLM istiyorsan Ollama.

Proje & Disiplin

Proje ne demek? Bir amaca yönelik dosyalar + context + bağlı ARM/Skill topluluğu. ~/.baios/projects// dizininde yaşar. Mimari

Kaç projem olabilir? Sınır yok. Yüzlerce proje açabilirsin.

Projeyi taşıyabilir miyim? Evet — baios proje export ZIP üretir, başka makinede baios proje import ile yüklenir.

Proje disiplini zorunlu mu? Hayır, opsiyonel. Ama çok faydalı — LLM'in sana önerileri projeye özel olur.

ARM & Skill

ARM nedir? Adaptör — BAIOS'u dış servislere (Gmail, GitHub, dosya, shell) bağlayan Python modülü. Katalog

Kaç ARM var? 28 core + community store'da 8 opsiyonel.

Skill ile ARM farkı?

  • ARM: tek iş yapar (bir API çağrısı, bir dosya işlemi)
  • Skill: birden fazla adımı birleştiren şablon
Kendi skill'imi nasıl yazarım? JSON dosyası ile. Geliştirici rehberi

Skill JSON'u arkadaşıma verebilir miyim? Evet — tek dosya, kopyala-yapıştır çalışır. Community store'a da yükleyebilirsin.

LLM

Hangi LLM'ler destekleniyor? OpenAI (GPT-4/4o/5), Anthropic (Claude Opus/Sonnet), Ollama (Llama 3, Mistral, Qwen), Local_4K (küçük modeller).

Provider değiştirmek? 

baios config set llm.provider anthropic
Konuşma kaldığı yerden devam eder — context korunur.

LLM'in uydurma (hallucination) sorunu olur mu? Context'te BAIOS'un güncel kapasiteleri enjekte edilir (SM-FIX1). LLM "bu skill yok" demez — listede var mı yok mu bilir. Yine de her önemli çıktıda insan onayı alınır.

Fallback nedir? Öncelikli provider çökerse sıradaki denenir. Kesintisiz hizmet.

Veri & Gizlilik

Verim nerede? Yerel: ~/.baios/. Dış servise sadece LLM çağrısı anında gider (sen OpenAI seçtiysen OpenAI'a).

BAIOS sunucusu var mı, verimi alır mı? Hayır. BAIOS tamamen yerel. Telemetri opsiyonel, varsayılan kapalı.

API key'lerim güvende mi? Vault'ta Fernet + PBKDF2 ile şifreli. Master password senin.

Yanlışlıkla rm -rf çalıştırır mı? Hayır. Shell ARM beyaz liste; rm kara listede. Genelde "onaysız riskli aksiyon yok" kuralı.

Otomasyon

Cron gibi mi? Evet + fazlası. Zaman, webhook, dosya değişimi, RSS, MQTT — hepsi tetikleyici olabilir.

Hata olursa? Her çalışma audit'e kayıt. Hata olursa e-posta/Telegram bildirimi (ayarlanabilir).

Otomasyonu durdur? 

baios> /otomasyon durdur

Entegrasyon

Slack/Discord/Telegram destekleniyor mu? Evet, 17 kanal hepsi. Bot kurulumu sırasıyla katalog → kanallar.

Notion'a bağlanabilir mi? Evet, notion ARM'ı ile.

MCP sunucusu eklenebilir mi? Evet, Birçok MCP aracı BAIOS'a ARM olarak bağlanabilir. MCP rehberi

GitHub Actions tetikleme? deploy ARM ile workflow_dispatch.

Kurumsal

Ekipte kullanılabilir mi? Aktif geliştirme — Faz 13 multi-tenant + RBAC + LDAP. Kurumsal

On-prem? Evet, docker-compose ile.

SOC2? 2027 hedefi. Compliance

Fiyat? Bireysel MIT (ücretsiz). Kurumsal destek + eğitim + özel geliştirme ayrı ticari anlaşma. İletişim

Edge

ESP32'de çalışır mı? Aktif araştırma. Köprü modu (ESP32 sensör + Pi orchestrator) önce hazır olacak. Edge

Raspberry Pi? Pi 4 / 5 için hedefte; henüz erken prototip.

Skill — Tekrarlanabilir İş Akışları

İçerik Üretimi

  • arastirma — konu web'de ara → LLM özetle → kaydet
  • blog_taslak — konu + kitle → 800-1200 kelime taslak
  • blog_post_taslak — (SM5) daha rafine blog taslağı
  • icerik_uretim — çok-adımlı içerik üretim
  • landing_copy_revize — landing metnini kitleye göre revize
  • url_ozet — URL → markdown → özet
  • gorsel_prompt — DALL-E için görsel prompt üret

Profesyonel İşler

  • muhasebe — fatura/CSV → analiz → rapor
  • fatura_olustur — kalem listesi → fatura PDF
  • harcama_takip — harcama kalemi ekle + kategori
  • veri_analiz — CSV oku → temel istatistik → görsel tarif
  • rakip_izleme — rakip site → değişiklik tespit
  • fiyat_takip — URL liste → fiyat izle → uyarı

Günlük

  • haber_ozet — RSS/feed → LLM özet → e-posta
  • gunluk_ozet — günün e-postalarını/mesajlarını özetle
  • toplanti_ozet — transkripti özetle
  • takip_mail — müşteri → takip e-postası taslağı
  • ilac_hatirlatma — zamanlı bildirim
  • dosya_organize — klasörü tipine göre düzenle

Geliştirici / Eğitim

  • flashcard — metin → Anki kart seti
  • konu_aciklama — konu → kademeli açıklama
  • git_changelog — git log → kullanıcı dostu değişiklik listesi

baios.dev (SM5)

  • site_qa — web sitesi kalite kontrolü (SEO, link, screenshot)
  • changelog_uret — commit aralığı → Türkçe changelog
  • release_notes — versiyon → release notes

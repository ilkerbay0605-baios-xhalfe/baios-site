Hızlı Başlangıç

🇬🇧 English

On dakikada BAIOS'u kurup ilk projeni açıp ilk skill'ini çalıştıracaksın.

1. Kurulum (1 dk)

# Erken sürüm — GitHub'dan kurulum (PyPI yayını yakında)
git clone https://github.com/ilkerbay0605-baios-xhalfe/baios
cd baios && ./run.sh start

İlk çalıştırmada kısa kişiselleştirme sorar (isim, dil). Tamam deyince ana menü açılır.

Web arayüzü: 

baios serve
# → http://localhost:8000

2. LLM Provider Seç (1 dk)

En az bir LLM provider gerekir. Üç seçenek:

OpenAI (API key varsa): 

baios config set llm.provider openai
baios config set openai.api_key sk-...

Anthropic: 

baios config set llm.provider anthropic
baios config set anthropic.api_key sk-ant-...

Yerel Ollama (internet yok, ücretsiz): 

# Ayrı pencerede:
ollama serve
ollama pull llama3.2

baios config set llm.provider local

Hassas veri varsa **local** öner. Hız ve doğruluk istersen OpenAI/Anthropic.

3. İlk Proje (2 dk)

Web UI: 1. Sol menü → Projeler → Yeni 2. Ad: denemem 3. Açıklama: BAIOS'u denemek için 4. Oluştur

CLI: 

baios> /proje yeni denemem "BAIOS'u denemek için"
baios> /proje sec denemem

Proje dizini oluşturuldu: ~/.baios/projects/denemem/.

4. Çalışma Disiplini Üret (2 dk)

Proje ne için? Disiplin Engine'e söyle, LLM sana özel iş akışı hazırlasın.

Web UI: 1. Proje sayfasında Çalışma Disiplini bölümüne gir 2. Hedef: "Haftalık bütçe tablosu tutmak ve ay sonu özet çıkarmak" 3. Üret

Sonuç discipline_v1.md — 3-5 fazlık adım adım yöntem, önerilen ARM ve skill'ler.

5. İlk Skill (2 dk)

Basit bir skill deneyelim — arastirma:

Web UI: 1. Sol menü → Kütüphane → Skill 2. arastirma bul → Çalıştır 3. konu: en iyi sabah rutinleri 4. Çalıştır

Üç adım çalışır: 1. Web araması 2. LLM özet 3. Dosyaya kaydet → ~/.baios/projects/denemem/outputs/arastirma_*.md

6. ARM Kur (opsiyonel, 1 dk)

Gmail okuma istersen gmail ARM'ını kur:

Web UI: 1. Sol menü → Kütüphane → ARM 2. gmail → Kur 3. Google hesap OAuth akışı 4. Tamam

Artık skill'lerde gmail kullanılabilir.

7. Chat'ten Konuş (1 dk)

Artık doğal dilde konuşabilirsin:

baios> bugünkü e-postalarımı özetle
  → gmail arm + llm_ozet skill
baios> istanbul hava durumu
  → weather arm
baios> bu araştırma dosyasından 10 flashcard çıkar
  → flashcard skill

BAIOS niyeti okur, uygun ARM/Skill'i çalıştırır. Riskli işlemde onay ister.

Sorun Giderme

Kurulum başarısız: 

# Python 3.12+ ve git gerekir
git clone https://github.com/ilkerbay0605-baios-xhalfe/baios
cd baios && ./run.sh start

Web UI açılmıyor: Port 8000 başka bir uygulamada olabilir. baios serve --port 8080

LLM hatası: 

baios config show       # mevcut provider'ı gör
baios config test llm   # bağlantı testi

Log: 

tail -f ~/.baios/logs/baios.log

Senaryolar

🇬🇧 English

BAIOS'un soyut tanımı yerine gerçek işlerde nasıl çalıştığı. Her senaryo kendi kullanıcısının gerçek bir gününden parça.

🎓 Öğretmen — Ders hazırlığı + öğrenci takibi

Hedef: Haftalık matematik dersini planla, yoklama tut, öğrenci dosyaları güncel olsun.

BAIOS kurgusu:

  • Proje: matematik-8-a
  • Bağlı ARM'lar: file, dosya_oku, calendar, gmail_send
  • Bağlı skill'ler: konu_aciklama, flashcard, gunluk_ozet
Tipik gün: 
baios> /proje sec matematik-8-a
baios> gelecek hafta trigonometri ders planı hazırla
  → konu_aciklama skill → docs/ders_plan_hafta14.md
baios> bu konudan 20 flashcard çıkar
  → flashcard skill → outputs/flashcards_trigonometri.csv
baios> ayşe'nin son sınav notunu dosyaya ekle 75
  → file arm → docs/ogrenciler/ayse.md güncel
baios> /otomasyon kur: her pazar 20:00 — hafta özetini bana e-posta at

📊 Muhasebeci — Fatura analizi + KDV kontrolü

Hedef: Ay sonu faturalarını kontrol et, KDV tutarlarını doğrula, rapor çıkar.

BAIOS kurgusu:

  • Proje: muhasebe-2026
  • ARM'lar: file, dosya_oku (PDF), sql (kayıt db)
  • Flow: muhasebe_kontrol
Tipik gün: 
baios> /proje sec muhasebe-2026
baios> bu ay gelen faturaları docs/faturalar-nisan/ içine koydum, kontrol et
  → dosya_oku (PDF) → llm analiz → tutarsızlık uyarısı
  → sql arm → ay toplamı hesapla → rapor.md
baios> şirket X için nisan özeti hazırla
  → veri_analiz skill → outputs/sirket_X_nisan.md

Audit: Her aksiyon ~/.baios/audit/ içinde kayıt — sonradan kim ne yaptı sorusuna cevap.

✍️ İçerik üretici — Blog + sosyal paylaşım

Hedef: Haftalık blog yazısı + Twitter/LinkedIn özetleri.

BAIOS kurgusu:

  • Proje: blog-kisilik-gelisimi
  • ARM'lar: brave_search, web_reader, file, notion
  • Skill'ler: arastirma, blog_post_taslak, url_ozet, gorsel_prompt
Tipik gün: 
baios> yeni konu: "sabah rutini bilimi"
baios> /skill calistir arastirma konu="sabah rutini bilimi"
  → brave_search + llm + file → docs/arastirma_sabah_rutini.md
baios> bu araştırmadan 1200 kelimelik blog post yaz, okuyucu: girişimci
  → blog_post_taslak skill → outputs/blog_sabah_rutini.md
baios> aynı konu için LinkedIn post taslağı (3 paragraf)
baios> blog'a görsel prompt öner
  → gorsel_prompt skill → dalle ile görsel üret

🔬 Veri bilimci — CSV keşfi + rapor

Hedef: Müşteri segmentasyon CSV'sini anla, ilk bulguları yaz, grafik öner.

BAIOS kurgusu:

  • Proje: musteri-segmentasyon-q2
  • ARM'lar: file, sql, http
  • Skill: veri_analiz
Tipik gün: 
baios> docs/customers.csv yükle
baios> /skill calistir veri_analiz dosya=docs/customers.csv
  → sql arm → temel istatistik
  → llm → bulgu yorumu
  → file → outputs/bulgular.md + grafik_tarif.md
baios> bu bulgulara göre 3 slayt taslağı
baios> /wiki olustur musteri-segmentasyon-bulgular  ← bilgi birikimi

🧑‍💻 Yazılımcı — baios.dev dogfood

Hedef: baios.dev web sitesini kendisi üzerinden yönet (bu sitenin gerçek hikayesi).

BAIOS kurgusu:

  • Proje: baios-dev (template'ten)
  • ARM'lar: git_ops, github, deploy, browser, web_reader, file
  • Skill'ler: landing_copy_revize, blog_post_taslak, changelog_uret, release_notes, site_qa
  • Flow: site_release
Tipik release: 
baios> /proje sec baios-dev
baios> yeni blog: "BAIOS neden ajan değil"
  → blog_post_taslak → outputs/blog_*.md (insan onayı)
baios> /flow calistir site_release versiyon=v2.6
  → changelog + release_notes → git branch → commit → push → PR aç
baios> PR merge'den sonra /arm deploy trigger main
baios> /arm deploy wait 300
baios> /skill calistir site_qa url=https://baios.dev

Detaylı hikaye →

🎓 Araştırmacı — Kaynak tarama + wiki birikimi

Hedef: Doktora tezi için "yapay zeka + hukuk" literatürü, sürekli artan wiki.

BAIOS kurgusu:

  • Proje: doktora-tez-ai-hukuk
  • ARM'lar: brave_search, web_reader, file, notion
  • Skill'ler: arastirma, url_ozet
  • Wiki otomatik üretimi
Tipik gün: 
baios> bu akademik makale URL'sini oku ve özetle: https://arxiv.org/abs/...
  → web_reader + llm_ozet → outputs/kaynak_N.md
baios> önemli bulgulardan biri: "AI karar verme şeffaflık sorunu"
  → /wiki olustur ai-karar-seffafligi
baios> geçen ay özetlediğim tüm kaynakları göster
  → file arm → docs/kaynaklar/ listele
baios> ilgili wiki sayfalarını LLM context'e ekle ve tez 3. bölümü için taslak hazırla

Kazanç: Her kaynak bilgiye kalıcı eklenir. Bir ay sonra LLM değişse bile birikim korunur.

Ortak Bağ: Disiplin

Altı senaryoda da farklı meslek, farklı iş. Ortak olan: 1. Proje — BAIOS her işi bir projede yaşatır. 2. Disiplin — Çalışma yöntemi dosyada, tekrarlanabilir. 3. Onay — Riskli işlem onay ister. 4. Context — LLM her zaman güncel duruma bakıyor. 5. Birikim — Her proje kendi bilgisini birikir, wiki büyür.

Senin senaryon?

Tutorial ile ilk projeyi aç → ve kendi disiplinini kur. Sorun çıkarsa sağ alttaki omgunbay sohbet penceresinden sor.
